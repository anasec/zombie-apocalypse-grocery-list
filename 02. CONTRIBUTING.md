# Contributing to the Zombie Apocalypse Grocery List

Thank you for helping improve this survival resource.

## How to Contribute

1. Fork the repository
2. Create a new branch for your changes
3. Add or update items in a way that is realistic and useful
4. Keep descriptions short, direct, and practical
5. Submit a Pull Request with a clear description of what you changed and why

## Guidelines

- Focus on realism and long-term survival
- No joke items
- No illegal items
- Keep lists simple and easy to read
- If adding a new year (2027+), copy the template from `FUTURE_YEARS_TEMPLATE.md`
